# Change Log

All notable changes to the **"witchblood-dark"** extension will be documented in this file.

## [1.0.0]-[1.0.3] - 2026-02-10

### Added
- **Initial Official Release**: Launch of the Witchblood Dark theme ecosystem.
- **Terminal-Inspired Aesthetic**: A high-contrast, deep-dark color palette inspired by Zsh (ohmyzsh/Darkblood) and iTerm2 terminal setups.
- **Bold Typography Support**: Optimized token colors for bold font weights to ensure maximum readability.

### Changed
- **Improved Contrast**: Adjusted Git decoration colors (Modified, Added, Deleted) to pop against the dark background without being distracting.
- **Bracket Pair Optimization**: Default bracket pair colorization has been streamlined to a unified color to maintain a clean, professional workspace.

### Fixed
- **Version Compatibility**: Resolved an engine error by lowering the minimum VS Code requirement to `^1.60.0`.
- **Language-Specific Highlights**: Fixed missing color tokens for JavaScript, CSS, and Python keywords.
